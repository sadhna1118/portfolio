## Setup Instructions

1. **Add Profile Picture**: 
   - Save the profile image as `assets/profile.jpg`
   - The image should be square (recommended: 500x500px or larger)

2. **Open the Portfolio**:
   - Simply open `index.html` in your web browser
   - No build process required - pure HTML, CSS, and JavaScript

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- SVG Icons

## Interactive Features

- **Sticky Navigation**: Navigation bar stays at the top while scrolling
- **Mobile Menu**: Hamburger menu for mobile devices
- **Scroll Animations**: Sections fade in as you scroll
- **Skill Bars**: Animated progress bars showing skill levels
- **Timeline**: Animated education timeline
- **Certificate Cards**: Hover effects on certificate cards
- **Smooth Scrolling**: Click navigation links for smooth page transitions

## Customization

To update the information:

1. Edit `index.html` to change content
2. Modify `styles.css` to adjust colors and styling
3. Update `script.js` for different interactive behaviors
